# Tatweer Accounting

![Version](https://img.shields.io/badge/version-6.1.2026-blue)


Customizations for ERPNext accounting module.

## Features

### Automated Depreciation
- Auto-approve and submit depreciation journal entries hourly
- Workflow: Draft → Approved → Submitted

### Root Trial Balance Report
- Trial balance for root-level accounts only
- Fiscal year filtering
- Multi-currency support
- Accounting dimensions support
